# Contributing to Awesome Hermes Skills

A curated collection of Hermes Agent skills — forked from [VoltAgent/awesome-openclaw-skills](https://github.com/VoltAgent/awesome-openclaw-skills) (MIT).

## Curation Criteria

- No spam/test skills
- English descriptions, min 3 words
- Security audited
- Hermes-compatible or easily adaptable
- MIT/Apache/BSD licensed

## Adding a Skill

### From OpenClaw/ClawHub:
```bash
python scripts/convert-openclaw-to-hermes.py /path/to/SKILL.md ./skills/
```

### Native Hermes Skill:
Follow the [Hermes skill authoring guide](https://hermes-agent.nousresearch.com/docs/skills/authoring).
